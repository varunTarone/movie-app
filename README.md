# Movie App
A React movie landing page built with OMDb API
<br>

## Tech Stack
- Vite
- React
- Node.js
- API ( movie database [OMDb API](https://www.omdbapi.com/) )
  
Hosted on [Netlify](https://moviefandom.netlify.app)
